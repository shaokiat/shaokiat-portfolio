# Portfolio Website

A modern, responsive portfolio demonstrating the productivity benefits of AI-powered development.

## Tech Stack
- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: GitHub Pages

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open:** [http://localhost:3000](http://localhost:3000)

## Antigravity Exploration

This project was created as an exploration with **Antigravity** (Google DeepMind's advanced agentic coding assistant). It serves as a demonstration of:
- **AI-Assisted Development**: Rapidly iterating on design and functionality using natural language prompts.
- **Workflow Automation**: Leveraging agentic capabilities to handle complex tasks like component structure, styling, and deployment configuration.
- **Human-AI Collaboration**: A testament to how AI tools can accelerate the transition from concept to deployed application for aspiring AI Engineers.
